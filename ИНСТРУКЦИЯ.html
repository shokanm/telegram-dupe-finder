Что делает программа

Сканирует две ваши Telegram-группы, находит одинаковые и похожие фотографии, и формирует HTML-отчёт с превью, датами и именами отправителей.

🔍
Точные дубликаты Один и тот же файл, отправленный несколько раз
🖼️
Похожие фото Визуально схожие изображения с оценкой схожести
📊
HTML-отчёт Открывается в браузере, всё наглядно
⚡️
Инкрементально Повторный запуск проверяет только новые фото
Первоначальная настройка
Один раз
1
Установите Python
  • Откройте в браузере: python.org/downloads
  • Скачайте Python 3.11 или новее
  • При установке поставьте галочку "Add Python to PATH"
2
Получите API-ключи Telegram
  1. Откройте в браузере: my.telegram.org
  2. Введите номер телефона → введите код из приложения Telegram
  3. Нажмите "API development tools"
  4. Заполните форму: App title — любое название, Platform — Desktop
  5. Нажмите Create application
  6. Скопируйте App api_id (число) и App api_hash (длинная строка)
3
Заполните файл config.py

Откройте файл config.py в папке программы (правой кнопкой → Открыть в TextEdit) и заполните:

config.py 
API_ID     = 1234567          # ← ваш api_id
API_HASH   = "abc123..."      # ← ваш api_hash
GROUP_1_ID = -10012345       # ← ID первой группы
GROUP_2_ID = -10067890       # ← ID второй группы
4
Узнайте ID ваших групп
  1. Откройте Терминал: Finder → Программы → Утилиты → Терминал
  2. Перейдите в папку программы:
Терминал 
cd путь/к/папке/telegram-dupe-finder
pip install -r requirements.txt
python list_groups.py
  1. Введите номер телефона и код из Telegram
  2. В списке найдите нужные группы, скопируйте их ID в config.py
Запуск программы

Откройте Терминал, перейдите в папку программы и выполните:

Терминал 
python main.py
🔑
Первый вход Программа попросит ввести номер телефона и код из Telegram. Это происходит только один раз — сессия сохраняется.
⏳
Первое сканирование Загружает все фото за последний год. Может занять 15–40 минут. Прогресс виден в терминале — не закрывайте окно до завершения.
⚡️
Повторные запуски Загружают только новые фото. Занимают несколько минут.
Просмотр результатов

После завершения программа покажет путь к файлу отчёта. Откройте его в браузере Safari или Chrome:

Путь к отчёту 
data/reports/report_2026-06-08_15-30.html
Точные

Одинаковые файлы, отправленные в группу несколько раз. Совпадение 100%.

Похожие

Визуально схожие изображения. Рядом отображается оценка схожести — чем меньше число, тем больше похожи.

Частые вопросы
Программа не запускается
Убедитесь что Python установлен: введите python --version в Терминале. Должна появиться версия, например Python 3.11.4. Если ошибка — переустановите Python с сайта python.org.
Ошибка "API_ID не заполнен"
Откройте файл config.py и убедитесь что все четыре поля заполнены: API_ID, API_HASH, GROUP_1_ID, GROUP_2_ID. Значение 0 означает что поле не заполнено.
Telegram просит код при каждом запуске
Не удаляйте файл data/session.session — в нём сохранена ваша авторизация. Если файл исчез, программа попросит войти заново — это нормально, нужно ввести номер и код один раз.
Как запустить повторно через неделю
Просто снова запустите python main.py. Программа автоматически определит, какие фото уже были проверены, и загрузит только новые. Повторные запуски занимают несколько минут.
Безопасность
⚠️
Важно: защитите эти файлы

Файлы config.py и data/session.session содержат доступ к вашему аккаунту Telegram. Не передавайте их посторонним и не публикуйте в интернете — через них можно получить доступ к вашим сообщениям.
